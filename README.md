## 📋 Overview

<p align="center">
  The SillyFox Pop-Up is a simple Firefox pop-up made with the sole purpose of making sure you accept your place and are content with it :3
</p>

<p align="center">
  <img src="images/firefox.png" width="400">
</p>

### Key Features
* 🛠️ **Easy to Configure**: You can configure the time between potential pop-ups, the chance of a pop-up, and even the image and text :D
* 🦊 **Silly**: A cute **and** *dominant* Firefox
* ⚡ **Can be annoying**: Duh

---

### Prerequisites
Ensure you have the Tampermonkey user script manager extension installed on your Firefox browser:
* [Tampermonkey](https://tampermonkey.net)
